Lino Galiana

Site personnel: https://linogaliana.netlify.com

Je suis statisticien à l’Insee où j’utilise des données de type big data ou des méthodes computationnelles pour étudier des phénomènes tels que les inégalités spatiales.

Je transforme la plupart de mes projets en packages R, disponibles sur ma page github ou gitlab. J’utilise fréquemment spark et python sur données volumineuses ou pour l’analyse spatiale (geopandas). Je fais également beaucoup de prosélytisme pour l’adoption systématique de git dans l’administration et la recherche.
